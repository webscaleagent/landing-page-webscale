# Expert Images

Place the expert profile images in this directory.

## Required Images

1. **benarab.png** - أ. سليم بن اعراب (الادارة والتسيير)
2. **abderrahim.jpg** - عبد الرحيم عبداللاوي (التسويق)
3. **chetta.png** - عبدالمالك شتى (تنظيم ورقمنة المؤسسات)
4. **noureddine-houari.jpg** - نورالدين هواري (استخدامات الذكاء الاصطناعي)

## Image Requirements

- Format: JPG or PNG
- Recommended size: 200x200 pixels minimum
- These will be displayed as circular avatars
- Professional headshots work best

## Usage

Images are referenced in the code as:
- `/experts/benarab.png`
- `/experts/abderrahim.jpg`
- `/experts/chetta.png`
- `/experts/noureddine-houari.jpg`

Files in the `public` folder are served at the root URL.
